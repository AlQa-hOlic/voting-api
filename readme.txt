Voting API

A sample voting API. The project can be run using the following command:

./gradlew bootRun

"Netty started on port 8080" indicates the server has started successfully and
is running on port 8080.

The project also includes unit tests which can be run using the following command:

./gradlew test

"BUILD SUCCESSFUL in 1s
4 actionable tasks: 4 up-to-date" indicates all the tests have passed.

The projects also has builtin API documentation using OpenAPI 3 and swagger UI. 
This can be accessed by opening the url "http://localhost:8080".

Checkout architecture.txt for more information on how the API is structured.
